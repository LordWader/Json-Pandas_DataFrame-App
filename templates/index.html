This program shows provides several methods to work with databases. You can choose through three options. Try it!

Select this option if you want to display information
about the parameters of all objects that interest you.
You can enter one column, several columns separated by commas,
or print all the information by typing the keyword: ALL

Select this option if you want to display information
about the parameters of a particular objects that interest you.
If the object has a parameter, it will be represented as 1,
if it is zero or EMPTY, then zero. Information is displayed
on a specific object. Other requests will be ignored!

Select this option if you want to display information about
the parameters of a particular object that interest you in
the form of a join line. Information is displayed for a specific
object, so for correct operation enter its number and
separator (default separator is a comma) Other requests will be ignored!

Select this option if you want to display the hash's of object's
For correct work enter its number, several numbers separated by commas or
print all hashes using keyword: ALL. Other requests will be ignored!
